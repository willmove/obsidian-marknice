# MarkNice WeChat 0.6.8

This release refreshes several WeChat layout themes, with a stronger focus on polished article headings and reliable pasted styling.

## What's New

- Added the **Warm Sun Red** theme, replacing the old Dream Purple slot with a cleaner red editorial style.
- Added numbered heading treatments for Warm Sun Red, including automatic section badges when headings are not already numbered.
- Refined the **Amber Orange** theme with centered label-style headings, custom ordered-list markers, and updated code, quote, image, table, and link styling.

## Improvements

- Renamed the Claude-inspired theme to **Warm Ceramic White** in user-facing copy.
- Migrates users whose default theme was Dream Purple to Warm Sun Red automatically.
- Updated README and publishing documentation to reflect the refreshed theme list.

## Compatibility

- Minimum Obsidian version: `1.5.0`.
- Existing notes and publishing settings continue to use the same conversion pipeline.
- Users with the old Dream Purple theme selected will be moved to Warm Sun Red on startup.

## Release Assets
