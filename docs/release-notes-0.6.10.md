# MarkNice WeChat 0.6.10

This release keeps the plugin package focused on runtime files while refining preview opening behavior.

## Improvements

- Improved preview view opening so the MarkNice preview leaf is revealed after it is created or reused.
- Updated mobile preview opening to request a new workspace leaf more reliably on Obsidian mobile.
- Moved the local public-account publishing introduction document out of version control and added it to `.gitignore`.

## Compatibility

- Minimum Obsidian version: `1.5.0`.
- Existing conversion, copy, export, OCR import, and WeChat draft publishing behavior remains unchanged.

## Release Assets

For Obsidian community plugin distribution, attach these files to the GitHub release tagged `0.6.10`:

- `manifest.json`
- `main.js`
- `styles.css`
